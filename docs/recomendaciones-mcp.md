# Recomendaciones para buscar MCPs

Si quieres explorar servidores MCP creados por la comunidad, una referencia útil es [mcpmarket.com/es](https://mcpmarket.com/es).

## Advertencia

No todos los MCPs publicados por la comunidad son seguros o apropiados para tu entorno. Antes de conectarte a uno, revisa al menos:

- Qué datos puede leer o modificar.
- Qué credenciales requiere.
- Si el mantenedor y el repositorio son confiables.
- Si el MCP pide permisos excesivos para su función.

## Criterio práctico

Usa MCPs comunitarios con el mismo criterio que cualquier dependencia externa: empieza con permisos mínimos, prueba en un proyecto de bajo riesgo y evita exponer secretos si no has validado el proveedor.
