# Recomendaciones para buscar skills

Si quieres descubrir skills creados por la comunidad, una referencia útil es [skills.sh](https://skills.sh/).

## Advertencia

No todos los skills publicados por la comunidad son seguros o adecuados para cualquier proyecto. Antes de instalar o usar uno, revisa al menos:

- Quién lo publicó.
- Qué permisos o accesos solicita.
- Si el contenido está actualizado y tiene mantenimiento.
- Si el objetivo del skill encaja con tu caso de uso.

## Criterio práctico

Usa skills comunitarios como apoyo, no como confianza implícita. Si un skill toca secretos, repositorios privados o flujos sensibles, valida su comportamiento primero en un entorno aislado.
