# Índice de recursos externos

Este directorio reúne referencias externas útiles para descubrir herramientas creadas por la comunidad.

## Qué revisar antes de usar algo de la comunidad

No asumas que una herramienta externa es segura solo por estar publicada. Antes de instalarla o conectarla, valida al menos:

- Quién la publica y si tiene mantenimiento activo.
- Qué permisos, datos o credenciales puede tocar.
- Si el alcance encaja con tu proyecto.
- Si puedes probarla primero en un entorno de bajo riesgo.

## Regla práctica

Usa estas referencias como punto de partida, no como señal de confianza implícita. Si una herramienta puede leer secretos, modificar repositorios o alcanzar servicios sensibles, revisa su comportamiento antes de adoptarla.
