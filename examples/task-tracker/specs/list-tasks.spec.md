# list-tasks

## Objetivo

Permitir listar todas las tareas registradas en el sistema.

## Ejemplos

### Caso feliz

- Dado que existen tareas registradas
- Cuando se solicita el listado
- Entonces se devuelven todas las tareas en el orden almacenado

### Sin datos

- Dado que no existen tareas registradas
- Cuando se solicita el listado
- Entonces se devuelve una lista vacía
