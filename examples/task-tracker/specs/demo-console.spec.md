# demo-console

## Objetivo

Mostrar en consola un flujo completo de creación y listado de tareas usando la estructura DDD del ejemplo.

## Ejemplos

### Caso feliz

- Dado que la aplicación inicia sin tareas
- Cuando la demo se ejecuta
- Entonces se crean dos tareas de ejemplo
- Y se imprime una línea por cada tarea en consola

### Criterio de salida

- Cada línea impresa debe contener `id`, `title` y `status` separados por `|`.
